# Intermediate Lessons

These lessons introduce classic data structures and algorithms after the beginner Python foundation.

| Lesson | File | Topic |
| --- | --- | --- |
| 01 | `01_linked_lists.py` | Nodes and linked lists |
| 02 | `02_insertion_sort.py` | Insertion sort |
| 03 | `03_merge_sort.py` | Merge sort |
| 04 | `04_quick_sort.py` | Quick sort |
| 05 | `05_radix_sort.py` | Radix sort for non-negative integers |
| 06 | `06_binary_trees.py` | Binary search trees and traversal |
| 07 | `07_avl_trees.py` | Self-balancing AVL trees |
| 08 | `08_dijkstra.py` | Shortest paths with Dijkstra's algorithm |

Run any lesson from the repository root:

```bash
python intermediate/01_linked_lists.py
```
