# Practice Exercises

Use these prompts after running the lesson files. Try solving each one before looking up an answer.

## Beginner

1. Change `1HelloWorld.py` so it asks for your name and prints a custom greeting.
2. In `2Variables.py`, create variables for your favorite food, favorite number, and whether you like Python.
3. In `3Math.py`, calculate the total cost of three items and print the result.
4. In `4Strings.py`, print the first and last character of a word.

## Data structures

1. In `7Lists.py`, add a new value to a list and print the updated list.
2. In `9Tuples.py`, create a tuple that stores a city and state, then unpack it into two variables.
3. In `10Dictionaries.py`, add a `hobby` key to the student dictionary.
4. In `15Comprehensions.py`, create a list of cubes from the numbers 1 through 5.

## Program flow

1. In `5Conditionals.py`, write an `if` statement that checks whether a score is passing.
2. In `6Loops.py`, write a loop that counts down from 5 to 1.
3. In `8Functions.py`, write a function that accepts a name and prints a greeting.
4. In `14Exceptions.py`, try converting user input to an integer and handle invalid input.

## Larger challenges

1. Build a small grade book using a list of dictionaries.
2. Save a list of favorite movies to a text file, then read it back.
3. Create a `Pet` class with `name`, `animal_type`, and `age` attributes.
4. Import `random` and make a number guessing game.
