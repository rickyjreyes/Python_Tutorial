# Beginner Lessons

These files extend the original beginner path with core Python topics that prepare learners for data structures and algorithms.

| Lesson | File | Topic |
| --- | --- | --- |
| 01 | `01_sets.py` | Sets, uniqueness, membership, and set operations |
| 02 | `02_classes_oop.py` | Classes, objects, attributes, methods, and simple OOP |
| 03 | `03_sorting.py` | Built-in sorting with `sorted()` and `.sort()` |

Run any lesson from the repository root:

```bash
python beginner/01_sets.py
```
