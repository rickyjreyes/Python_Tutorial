# Beginner Lessons

These lessons form the main beginner path from easiest to hardest. The files use zero-padded lesson numbers so GitHub displays them in order.

| Lesson | File | Topic |
| --- | --- | --- |
| 01 | `01_hello_world.py` | Printing, comments, and basic input |
| 02 | `02_variables.py` | Variables, data types, assignment, and scope |
| 03 | `03_math.py` | Arithmetic, built-in math helpers, and randomness |
| 04 | `04_strings.py` | String indexing, slicing, formatting, and methods |
| 05 | `05_conditionals.py` | Boolean comparisons and `if`/`else` statements |
| 06 | `06_loops.py` | `while` loops, `for` loops, and loop control variables |
| 07 | `07_lists.py` | Lists, indexing, slicing, and updating values |
| 08 | `08_tuples.py` | Tuple creation and immutability |
| 09 | `09_dictionaries.py` | Dictionaries, key/value access, and looping over data |
| 10 | `10_sets.py` | Sets, uniqueness, membership, and set operations |
| 11 | `11_functions.py` | Defining functions, parameters, and return values |
| 12 | `12_file_io.py` | Reading and writing text files with `pathlib` |
| 13 | `13_classes_oop.py` | Classes, objects, attributes, methods, and simple OOP |
| 14 | `14_modules.py` | Importing standard library modules and helper functions |
| 15 | `15_exceptions.py` | Handling runtime errors with `try`/`except` |
| 16 | `16_comprehensions.py` | Building lists and dictionaries with comprehensions |
| 17 | `17_sorting.py` | Built-in sorting with `sorted()` and `.sort()` |

Run any lesson from the repository root:

```bash
python beginner/01_hello_world.py
```
